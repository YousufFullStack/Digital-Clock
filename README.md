# ⏰ Digital Clock

A simple and responsive digital clock built using **HTML**, **CSS**, and **JavaScript**. The clock displays the current local time and updates automatically every second.

## 🚀 Features

- Displays the current local time
- Updates every second in real time
- Responsive and modern UI
- Built with vanilla JavaScript (no external libraries)

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
